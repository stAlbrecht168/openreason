# Custom GPT Setup

This folder explains how to adapt OpenReason for a Custom GPT.

## Instructions

Use `custom-gpt/instructions.md` as the base instruction file.

## Knowledge files

Upload:

- `README.md`
- `docs/MANIFESTO.md`
- `docs/GETTING_STARTED.md`
- all files in `frameworks/`
- representative example analyses

## Limitation

A Custom GPT cannot automatically run the TypeScript reference implementation. It can still use the framework files and evidence model as guidance.
