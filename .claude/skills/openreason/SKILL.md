# OpenReason Skill for Claude Code

Use this skill when the user asks to analyze arguments, videos, political statements, rhetoric, discourse, framing, or group-based comparisons.

Workflow:

1. Identify the user intent.
2. Load relevant frameworks from `frameworks/`.
3. Use the evidence-status model.
4. Generate or inspect an OpenReason analysis packet.
5. Produce a final report that separates observation, inference, interpretation, social effects, and hypotheses.

Always avoid unsupported motive attribution.
