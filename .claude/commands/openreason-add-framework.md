# /openreason-add-framework

Add a new OpenReason framework.

Required steps:

1. Read `docs/FRAMEWORK_AUTHORING.md`.
2. Create a YAML file under `frameworks/<domain>/<id>.yaml`.
3. Include purpose, intents, triggers, concepts, evidence statuses, steps, rules, questions, outputs, limitations, and references.
4. Add or update tests.
5. Run validation and tests.
6. Summarize limitations and verification status.
